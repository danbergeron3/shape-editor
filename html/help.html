Description

Paste clipboard items.
CTRL-D

CTRL-F
End poly line drawing.
Mouse Wheel
Zoom in and out.
Left Click
To select item.
Hold and drag for multi-select.
Pan Button
toggles pan, disables select, allows movement around canvas.
Sketch Button
Toggles sketch on and off
Color Button
Allows selection of color through hex code or picker.
Shape Button
Allows the drawing of various shapes.
Can adjust line thickness.
Grid Button
Toggles Grid view.
Snap to grid button
Toggles snap to grid.
Clear button
Resets canvas.

2. JPEG
